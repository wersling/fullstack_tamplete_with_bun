## 📦 技术栈速览

### 后端
- **Runtime**: Bun - 高性能 JavaScript 运行时
- **Framework**: Hono - 轻量级 Web 框架
- **Auth**: Better Auth - 现代认证解决方案
- **ORM**: Drizzle ORM - TypeScript ORM
- **Database**: SQLite (可切换到 PostgreSQL/MySQL)

### 前端
- **Build Tool**: Vite - 快速构建工具
- **Library**: React 19 - UI 框架
- **Routing**: React Router v7 - 路由管理
- **Styling**: Tailwind CSS v4 - 原子化 CSS
- **UI**: Shadcn/ui - 现代 UI 组件库
